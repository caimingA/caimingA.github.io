蒙特卡罗与MCMC

math RL ML
math basic
Publish Date:   2022-06-18

简述

蒙特卡罗方法是一种用多次取样来逐渐逼近样本真实分布的方法。

另外蒙特卡罗并不是人的名字，而是位于摩纳哥的一个赌场的名字。

简单的例子

最经典的例子应该是几何概型中的算$\pi$的数值。也就是高中课本上的撒豆子问题。

通过多次随机取样，取样点会随机分布在一个正方形的区域里，然后在该区域中画圆，就会是样本分成两部分。一部分在圆内，另一部分在圆外。然后通过算圆内的样本数占总样本数的比例可以近似算的圆的面积，最后通过圆面积公式能算得$\pi$的取值。

实操

sampleNum = 10000
counter = 0
plt.figure(figsize=(8, 8))

y1 = list()
x2 = list()

    x = random.random()
    y = random.random()
    if x**2 + y**2 < 1:
        counter += 1
        x1.append(x)
        y1.append(y)
    else:
        x2.append(x)
        y2.append(y)
   
print(4*counter/sampleNum)

plt.scatter(x1, y1, color='r')

拒绝采样

重要性采样是对目标函数$f(x)$添加权重，但我们的得到的样本$\{x_{i}\}_{N}$是服从建议分布的，而不是服从目标分布。
那我们是否有办法真正的得到从目标分布中采样的来的样本呢？
答案是肯定的。

MCMC

拒绝采样适用于低维简单分布，而 MCMC 专为高维复杂分布设计，通过构建马尔可夫链使其平稳分布收敛于目标分布。
MCMC采样很神奇的点在于，每次采样的时候不是完全独立，而是上一个采样的样本会对下一采样的样本有一定约束。

一个马尔可夫链拥有唯一平稳分布通常需要具备一下三点性质：

  • 不可约性（Irreducible）：系统中的任何一个状态都可以通过有限步转移到达其他所有状态。
  • 非周期性（Aperiodic）：系统从某状态出发，返回该状态的可能步数不具有公约数，避免了系统出现死循环振荡。
  • 正常返性（Positive Recurrent）：系统平均返回到某一状态的时间是有限的。

一言蔽之就是说，通过调整转移矩阵的对角线元素可以完成归一化，并且不影响细致平稳条件的成立。

Metropolis算法

接下来写一下到目前为止的算法。
我们的目标实在满足细致平衡条件的$\widetilde{\boldsymbol{Q}}$上采样，如果$\widetilde{\boldsymbol{Q}}$比较难以采样，我们便在容易采样的$\boldsymbol{Q}$上采样。
注意，$\widetilde{q}_{ji}$和${q}_{ji}$差一个$\alpha_{ji}$的倍数。
当我们通过${q}_{ji}$从状态$i$选择状态$j$的时候，如果$i \ne j$，我们会发现我们实际上需要的选择的概率为$\tilde{q}_{ji}$。
与拒绝采样相似，$\alpha_{ji}$可以被看作是一个接受概率。
换言之，当我们处于$i$状态，而下一个被选择的状态是与$i$不同的$j$的时候，我们以概率$\alpha_{ji}$接受这个选择，以$1-\alpha_{ji}$的概率拒绝这个选择。
当我们拒绝选择状态$j$的时候，我们会留在状态$i$，即，有采样了一次$i$。

需要注意的是，我们由状态$i$到状态$i$时的接受率并不是$\alpha_{ii}$而是$1-\alpha_{ji}$。
这就是之前所讲的，通过调整转移矩阵的对角线元素可以完成归一化，实现最终以$p_{ji}$进行转移。

最后，Metropolis算法总结如下

Metropolis采样

初始状态的$x_{0}$, t时刻状态为$x_{t}$。
通过如下流程采样样本$x_{t+1}$:

  1. 通过$q_{t+1, t}$采样一个样本$x^{*}_{t+1}$。
  2. 从$\mathrm{Unif}(0, 1)$中，生成一个随机数$u$。
  3. 如果$u \le \alpha_{t+1, t}$，$x_{t+1} \gets \alpha_{t+1, t} \cdot x^{*}_{t+1}$；否则，$x_{t+1} \gets x_{t}$。

这个算法与拒绝采样相似，但不完全一直。
拒绝采样中如果样本被拒绝那就是拒绝了，但Metropolis算法的话，会把原地踏步把目前的状态计入时刻$t$的样本。

Metropolis-Hastings 采样

初始状态的$x_{0}$, t时刻状态为$x_{t}$。
通过如下流程采样样本$x_{t+1}$:

  1. 通过$q_{t+1, t}$采样一个样本$x^{*}_{t+1}$。
  2. 从$\mathrm{Unif}(0, 1)$中，生成一个随机数$u$。
  3. 如果$u \le \mathcal{A}_{t+1, t}=\mathrm{min}(1, \frac{\pi_{t+1}q_{t, t+1}}{\pi_{t}q_{t+1, t}})$，$x_{t+1} \gets \alpha_{t+1, t} \cdot x^{*}_{t+1}$；否则，$x_{t+1} \gets x_{t}$。

一段M-H采样高斯分布的python 代码

import random
import math
from scipy.stats import norm
import matplotlib.pyplot as plt

def norm_dist_prob(theta):
    y = norm.pdf(theta, loc=10, scale=5)
    return y

T = 5000
pi = [0 for i in range(T)]
sigma = 1
t = 0
while t < T-1:
    t = t + 1
    pi_star = norm.rvs(loc=pi[t - 1], scale=sigma, size=1, random_state=None)   #状态转移进行随机抽样
    alpha = min(1, (norm_dist_prob(pi_star[0]) / norm_dist_prob(pi[t - 1])))   #alpha值

    u = random.uniform(0, 1)
    if u < alpha:
        pi[t] = pi_star[0]
    else:
        pi[t] = pi[t - 1]


plt.scatter(pi, norm.pdf(pi, loc=10, scale=5),label='Target Distribution', c= 'red')
num_bins = 50
plt.hist(pi, num_bins, density=1, facecolor='green', alpha=0.7,label='Samples Distribution')
plt.legend()
plt.show()

吉布斯采样（Gibbs Sampling）

接下来，我们看看更现实的应用场景，高维数据的情况。

假设$\boldsymbol{X}_{t}= \{X_{t, 1}, X_{t, 2}, \cdots, X_{t, d}\}$是一个$d$维的随机变量。
Gibbs采样每个时刻$t$，只对$d$维中的一维进行采样。
比如说我们接下来要对第$h$维采样，我们将时刻$t+1$的样本记作$\boldsymbol{X}_{t+1} = \boldsymbol{X}_{t, [X_{t, h}=X_{t+1, h}]}$
，我们定义转移概率（这里的多维变量，我们用最稳妥的写法）为

其中$\boldsymbol{X}_{t, -h} = \boldsymbol{X}_{t} \setminus \{X_{t, h}\}$。
这个式子意味着，我们每次固定第$h$维以外的维度的取值，然后只更新第$h$维的取值。
然后，我们定义接受率

所以，对于每一个时刻$t$所作的选择，Gibbs采样都不做拒绝地将其作为样本保留下来。

Gibbs采样

初始状态为$\boldsymbol{x}_{0}$, t时刻状态的分布为$\boldsymbol{x}_{t}$。
通过如下流程采样样本$\boldsymbol{x}_{t+1}$:

  1. 均匀的从$[d]$中选取一个维度$h$。
  2. 计算$q(\boldsymbol{X}_{t+1} \mid \boldsymbol{X}_{t})= \frac{P(\boldsymbol{X}_{t, [X_{t, h}=X_{t, h}]})}{P(\boldsymbol{X}_{t, -h})}$
  3. 通过$q(\boldsymbol{X}_{t+1} \mid \boldsymbol{X}_{t})$采样第$h$维的一个状态$x_{t+1, h}$。
  4. $x_{t+1} \gets \boldsymbol x_{t, [x_{t, h}=x_{t+1, h}]}$。

最后一步，是将第$d$维的数据替换成新采样的状态。

import random
import math
from scipy.stats import beta
from scipy.stats import norm
import matplotlib.pyplot as plt

from mpl_toolkits.mplot3d import Axes3D
from scipy.stats import multivariate_normal

samplesource = multivariate_normal(mean=[5,-1], cov=[[1,0.5],[0.5,2]])

def p_ygivenx(x, m1, m2, s1, s2):
    return (random.normalvariate(m2 + rho * s2 / s1 * (x - m1), math.sqrt(1 - rho ** 2) * s2))

def p_xgiveny(y, m1, m2, s1, s2):
    return (random.normalvariate(m1 + rho * s1 / s2 * (y - m2), math.sqrt(1 - rho ** 2) * s1))

N = 5000
K = 20
x_res = []
y_res = []
z_res = []
m1 = 5
m2 = -1
s1 = 1
s2 = 2

rho = 0.5
y = m2

for i in range(N):
    for j in range(K):
        x = p_xgiveny(y, m1, m2, s1, s2)   #y给定得到x的采样
        y = p_ygivenx(x, m1, m2, s1, s2)   #x给定得到y的采样
        z = samplesource.pdf([x,y])
        x_res.append(x)
        y_res.append(y)
        z_res.append(z)

num_bins = 50
plt.scatter(x_res, norm.pdf(x_res, loc=5, scale=1),label='Target Distribution x', c= 'green')
plt.scatter(y_res, norm.pdf(y_res, loc=-1, scale=2),label='Target Distribution y', c= 'orange')
plt.hist(x_res, num_bins, density=1, facecolor='Cyan', alpha=0.5,label='x')
plt.hist(y_res, num_bins, density=1, facecolor='magenta', alpha=0.5,label='y')
plt.title('Histogram')
