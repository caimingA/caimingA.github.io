蒙特卡罗与MCMC

math RL ML
math basic
Publish Date:   2022-06-18

简述

蒙特卡罗方法是一种用多次取样来逐渐逼近样本真实分布的方法。

另外蒙特卡罗并不是人的名字，而是位于摩纳哥的一个赌场的名字。

简单的例子

最经典的例子应该是几何概型中的算π\piπ的数值。也就是高中课本上的撒豆子问题。

通过多次随机取样，取样点会随机分布在一个正方形的区域里，然后在该区域中画圆，就会是样本分成两部分。一部分在圆内，另一部分在圆外。然后通过算圆内的样本数占总样本数的比例可以近似算的圆的面积，最后通过圆面积公式能算得π\piπ的取值。

重要性采样

有的时候直接从目标分布（注意，这个分布是已知的）采样非常困难，代价高昂，或者一定样本的到的精度不高时。 我们可以引入一个容易采样的建议分布（Proposal Distribution），并赋予每个样本相应的权重，来无偏地估计目标分布的期望值。

假设我们选择的易于采样的建议分布的pdf为q(x)q(x)q(x)，

Ex∼π(x)[f(x)]=∫xπ(x)f(x)dx=∫xq(x)π(x)q(x)f(x)dx=Ex∼q(x)[π(x)q(x)f(x)].\begin{aligned} \mathbb{E}_{x\sim \pi(x)}[f(x)] &= \int_{x}\pi(x)f(x) dx\\ &= \int_{x}q(x)\frac{\pi(x)}{q(x)}f(x) dx \\ &= \mathrm{E}_{x \sim q(x)}\left[\frac{\pi(x)}{q(x)}f(x)\right]. \end{aligned} Ex∼π(x)​[f(x)]​=∫x​π(x)f(x)dx=∫x​q(x)q(x)π(x)​f(x)dx=Ex∼q(x)​[q(x)π(x)​f(x)].​

如果我们把π(x)q(x)\frac{\pi(x)}{q(x)}q(x)π(x)​定义为w(x)w(x)w(x)，那么

Ex∼q(x)[π(x)q(x)f(x)]=Ex∼q(x)[w(x)f(x)]\mathrm{E}_{x \sim q(x)}\left[\frac{\pi(x)}{q(x)}f(x)\right] = \mathrm{E}_{x \sim q(x)}\left[w(x)f(x)\right] Ex∼q(x)​[q(x)π(x)​f(x)]=Ex∼q(x)​[w(x)f(x)]

可以看作我们在建议分布上采样的时候，目标函数f(x)f(x)f(x)的值要乘上一个权重w(x)w(x)w(x)。 对于不同的xxx，权重不同，说明对应的f(x)f(x)f(x)的重要性不同，这个便是该小节标题中“重要性”的来源。

对XXX进行采样之后，再次根据大数定律，

Ex∼q(x)[w(x)f(x)]=1N∑iNw(xi)f(xi).\mathrm{E}_{x \sim q(x)}\left[w(x)f(x)\right] = \frac{1}{N}\sum_{i}^{N}w(x_{i})f(x_{i}). Ex∼q(x)​[w(x)f(x)]=N1​i∑N​w(xi​)f(xi​).

与上面的直观理解相似，“重要性”体现在，对于不同xix_{i}xi​，权重w(xi)w(x_{i})w(xi​)不同，f(xi)f(x_{i})f(xi​)的”贡献“自然不同。 对上面的式子中，我们需要注意的是，目标分布π(x)\pi(x)π(x)，我们选取的建议分布q(x)q(x)q(x)和目标函数f(x)f(x)f(x)都是已知的。我们只是想通过采样之后得到f(x)f(x)f(x)的期望。

拒绝采样

重要性采样是对目标函数f(x)f(x)f(x)添加权重，但我们的得到的样本{xi}N\{x_{i}\}_{N}{xi​}N​是服从建议分布的，而不是服从目标分布。 那我们是否有办法真正的得到从目标分布中采样的来的样本呢？ 答案是肯定的。

因为目标分布的直接采样困难， 在拒绝采样的过程中，我们还是要先选取一个易于采样的建议分布q(x)q(x)q(x)。 与重要性采样不同的是，我们还要选取一个常数MMM，使得

M⋅q(x)≥π(x).M\cdot q(x) \ge \pi(x). M⋅q(x)≥π(x).

自然，我们接下来应该做的是直接从q(x)q(x)q(x)中采样。 在重要性采样中，我们对于没一个采到的样本都保留下来，进行利用。 而在拒绝采样中，对于采到的样本xix_{i}xi​，我们以αaccept\alpha_{\mathrm{accept}}αaccept​的概率进行保留。自然以αreject=1−αaccept\alpha_{\mathrm{reject}} = 1 - \alpha_{\mathrm{accept}}αreject​=1−αaccept​的概率丢弃。

对于xix_{i}xi​接受概率的计算：

αaccept=π(xi)M⋅q(xi).\alpha_{\mathrm{accept}} = \frac{\pi(x_{i})}{M\cdot q(x_{i})}. αaccept​=M⋅q(xi​)π(xi​)​.

为了实现概率接受和概率丢弃，我们可以随机生成一个服从Unif(0,1)\mathrm{Unif}(0, 1)Unif(0,1)的随机数uuu，如果uuu落在[0,αaccept][0, \alpha_{\mathrm{accept}}][0,αaccept​]，xix_{i}xi​就会被接受，反之，如果u∈(αaccept,1]u \in (\alpha_{\mathrm{accept}}, 1]u∈(αaccept​,1]，xix_{i}xi​便会被丢弃。

MCMC

拒绝采样适用于低维简单分布，而 MCMC 专为高维复杂分布设计，通过构建马尔可夫链使其平稳分布收敛于目标分布。 MCMC采样很神奇的点在于，每次采样的时候不是完全独立，而是上一个采样的样本会对下一采样的样本有一定约束。

马尔可夫链

对于一组随机变量X={Xt}t≥0\boldsymbol{X} = \{X_{t}\}_{t \ge 0}X={Xt​}t≥0​，其中ttt代表时刻，如果有

P(Xt∣{Xt}0≤t≤t−1)=P(Xt∣Xt−1),P(X_{t} \mid \{X_{t}\}_{0 \le t \le t-1}) = P(X_{t} \mid X_{t-1}), P(Xt​∣{Xt​}0≤t≤t−1​)=P(Xt​∣Xt−1​),

那么，称X={Xt}t≥0\boldsymbol{X} = \{X_{t}\}_{t \ge 0}X={Xt​}t≥0​为马尔可夫链。 进一步，P(Xt∣Xt−1)P(X_{t}\mid X_{t-1})P(Xt​∣Xt−1​)被称为Xt−1X_{t-1}Xt−1​到XtX_{t}Xt​的转移概率分布。

例如，我们在这里假设X≤t={Xt}≤t∈[1,2,3](t+1)\boldsymbol{X}_{\le t} = \{X_{t}\}_{\le t} \in [1, 2, 3]^{(t+1)}X≤t​={Xt​}≤t​∈[1,2,3](t+1)， 我们定义

其中，Pji=P(Xt=j∣Xt−1=i)P_{ji} = P(X_{t}=j \mid X_{t-1}=i)Pji​=P(Xt​=j∣Xt−1​=i)。 我们称上述的矩阵为转移概率矩阵。

我们进一步定义

π(t)=[π1(t),π2(t),π3(t)]⊤,\pi(t) = [\pi_{1}(t), \pi_{2}(t), \pi_{3}(t)]^{\top}, π(t)=[π1​(t),π2​(t),π3​(t)]⊤,

π(t)⊤=π(0)⊤Pt.\pi(t)^{\top} = \pi(0)^{\top}P^{t}. π(t)⊤=π(0)⊤Pt.

我们都知道，对于一个转移过程，若

π(t)⊤=π(t)⊤P，\pi(t)^{\top} = \pi(t)^{\top}P， π(t)⊤=π(t)⊤P，

则称π(t)\pi(t)π(t)是马尔可夫链的平稳分布。

一个马尔可夫链拥有唯一平稳分布通常需要具备一下三点性质：

  • 不可约性（Irreducible）：系统中的任何一个状态都可以通过有限步转移到达其他所有状态。
  • 非周期性（Aperiodic）：系统从某状态出发，返回该状态的可能步数不具有公约数，避免了系统出现死循环振荡。
  • 正常返性（Positive Recurrent）：系统平均返回到某一状态的时间是有限的。

最后，我们定义一种特殊的马尔可夫链。

对于任意时刻ttt，如果一个马尔可夫链满足细致平衡方程（detailed balance equation）:

∀i∀j,pjiπi=pijπj,\forall i \forall j, \qquad p_{ji}\pi_{i} = p_{ij}\pi_{j}, ∀i∀j,pji​πi​=pij​πj​,

则称该马尔可夫链为可逆马尔可夫链。

定理 1
满足细致平衡方程的状态分布π\piπ是该马尔可夫链的平稳分布。

MCMC采样

如定理1所言，我们只要能找到一个满足细致平衡方程的P\boldsymbol{P}P，那么，马尔可夫转移每次选择的状态（在本文中为1 or 2 or 3）可以作为采的样本，并且该样本符合目标分布π\piπ。

接下来的问题变成如何去寻找满足条件的P\boldsymbol{P}P。

我们可以先任取一个转移矩阵Q\boldsymbol{Q}Q。此时我们一般有

其中αji\alpha_{ji}αji​和αij\alpha_{ij}αij​为缩放系数。

进一步，要是等式恒成立，我们可以简单地令

αji=πjqij,αij=πiqji.\alpha_{ji} = \pi_{j}q_{ij}, \qquad \alpha_{ij} = \pi_{i}q_{ji}. αji​=πj​qij​,αij​=πi​qji​.

那么，我们是否还可以用Q~\widetilde{\boldsymbol{Q}}Q​来进行采样呢？ 答案是可以的。 实际上，我们可以这么定义pjip_{ji}pji​和q~ji\tilde{q}_{ji}q~​ji​:

至于这样的pjip_{ji}pji​是否满足细致平稳呢？ 我们发现，对于任意πi\pi_{i}πi​，πj\pi_{j}πj​，fi,jf_{i, j}fi,j​，和fj,if_{j,i}fj,i​都有

一言蔽之就是说，通过调整转移矩阵的对角线元素可以完成归一化，并且不影响细致平稳条件的成立。

Metropolis算法

接下来写一下到目前为止的算法。 我们的目标实在满足细致平衡条件的Q~\widetilde{\boldsymbol{Q}}Q​上采样，如果Q~\widetilde{\boldsymbol{Q}}Q​比较难以采样，我们便在容易采样的Q\boldsymbol{Q}Q上采样。 注意，q~ji\widetilde{q}_{ji}q​ji​和qji{q}_{ji}qji​差一个αji\alpha_{ji}αji​的倍数。 当我们通过qji{q}_{ji}qji​从状态iii选择状态jjj的时候，如果i≠ji \ne ji=j，我们会发现我们实际上需要的选择的概率为q~ji\tilde{q}_{ji}q~​ji​。 与拒绝采样相似，αji\alpha_{ji}αji​可以被看作是一个接受概率。 换言之，当我们处于iii状态，而下一个被选择的状态是与iii不同的jjj的时候，我们以概率αji\alpha_{ji}αji​接受这个选择，以1−αji1-\alpha_{ji}1−αji​的概率拒绝这个选择。 当我们拒绝选择状态jjj的时候，我们会留在状态iii，即，有采样了一次iii。

需要注意的是，我们由状态iii到状态iii时的接受率并不是αii\alpha_{ii}αii​而是1−αji1-\alpha_{ji}1−αji​。 这就是之前所讲的，通过调整转移矩阵的对角线元素可以完成归一化，实现最终以pjip_{ji}pji​进行转移。

  1. 通过qt+1,tq_{t+1, t}qt+1,t​采样一个样本xt+1∗x^{*}_{t+1}xt+1∗​。
  2. 从Unif(0,1)\mathrm{Unif}(0, 1)Unif(0,1)中，生成一个随机数uuu。
  3. 如果u≤αt+1,tu \le \alpha_{t+1, t}u≤αt+1,t​，xt+1←αt+1,t⋅xt+1∗x_{t+1} \gets \alpha_{t+1, t} \cdot x^{*}_{t+1}xt+1​←αt+1,t​⋅xt+1∗​；否则，xt+1←xtx_{t+1} \gets x_{t}xt+1​←xt​。

这个算法与拒绝采样相似，但不完全一直。 拒绝采样中如果样本被拒绝那就是拒绝了，但Metropolis算法的话，会把原地踏步把目前的状态计入时刻ttt的样本。

Metropolis-Hastings 采样

通过αji\alpha_{ji}αji​来接受拒绝下一个状态的选择其实会面临一个问题。 如果αji\alpha_{ji}αji​太小，那采样会一直原地踏步，无法对其他状态进行探索。

为了解决这个问题，我们先回到细致平稳方程

πiqjiαji=πjqijαij.\pi_{i}q_{ji}\alpha_{ji} = \pi_{j}q_{ij}\alpha_{ij}. πi​qji​αji​=πj​qij​αij​.

正如我前面所说，αji\alpha_{ji}αji​和αij\alpha_{ij}αij​是一个缩放，那么我们能否在这个式子的基础上，左右两边再同乘上一个常数cic_{i}ci​，让ci⋅αjic_{i}\cdot \alpha_{ji}ci​⋅αji​来作为接受率呢？ 显然是可以的，但是接受率一定要是一个介于0到1之间的数，我们再乘上cic_{i}ci​之后，ci⋅αjic_{i}\cdot \alpha_{ji}ci​⋅αji​和ci⋅αijc_{i}\cdot \alpha_{ij}ci​⋅αij​都不能超多一。 对于ci⋅αjic_{i}\cdot \alpha_{ji}ci​⋅αji​，我们何时可以将它缩放到最大呢？ 当ci⋅αji=1c_{i}\cdot \alpha_{ji} = 1ci​⋅αji​=1或者ci⋅αij=1c_{i}\cdot \alpha_{ij}=1ci​⋅αij​=1的时候。

令Aji:=ci⋅αji\mathcal{A}_{ji}:=c_{i}\cdot \alpha_{ji}Aji​:=ci​⋅αji​，则有

Aji={1,αji≥αij,πjqijπiqji,αji<αij.\begin{aligned} \mathcal{A}_{ji} = \left\{ \begin{array}{ll} 1, & \alpha_{ji} \ge \alpha_{ij},\\ \frac{\pi_{j}q_{ij}}{\pi_{i}q_{ji}}, & \alpha_{ji} < \alpha_{ij}. \end{array} \right. \end{aligned} Aji​={1,πi​qji​πj​qij​​,​αji​≥αij​,αji​<αij​.​​

最后，我们只需要让新的接受率

Aji=min(1,πjqijπiqji)\mathcal{A}_{ji} = \mathrm{min}(1, \frac{\pi_{j}q_{ij}}{\pi_{i}q_{ji}}) Aji​=min(1,πi​qji​πj​qij​​)

就好。

如果，qij=qjiq_{ij} = q_{ji}qij​=qji​， 则

Aji=min(1,πjπi)\mathcal{A}_{ji} = \mathrm{min}(1, \frac{\pi_{j}}{\pi_{i}}) Aji​=min(1,πi​πj​​)

通过如下流程采样样本xt+1x_{t+1}xt+1​:

  1. 通过qt+1,tq_{t+1, t}qt+1,t​采样一个样本xt+1∗x^{*}_{t+1}xt+1∗​。
  2. 从Unif(0,1)\mathrm{Unif}(0, 1)Unif(0,1)中，生成一个随机数uuu。
  3. 如果u≤At+1,t=min(1,πt+1qt,t+1πtqt+1,t)u \le \mathcal{A}_{t+1, t}=\mathrm{min}(1, \frac{\pi_{t+1}q_{t, t+1}}{\pi_{t}q_{t+1, t}})u≤At+1,t​=min(1,πt​qt+1,t​πt+1​qt,t+1​​)，xt+1←αt+1,t⋅xt+1∗x_{t+1} \gets \alpha_{t+1, t} \cdot x^{*}_{t+1}xt+1​←αt+1,t​⋅xt+1∗​；否则，xt+1←xtx_{t+1} \gets x_{t}xt+1​←xt​。

一段M-H采样高斯分布的python 代码

import random
import math
from scipy.stats import norm
import matplotlib.pyplot as plt

def norm_dist_prob(theta):
    y = norm.pdf(theta, loc=10, scale=5)
    return y

T = 5000
pi = [0 for i in range(T)]
sigma = 1
t = 0
while t < T-1:
    t = t + 1
    pi_star = norm.rvs(loc=pi[t - 1], scale=sigma, size=1, random_state=None)   #状态转移进行随机抽样
    alpha = min(1, (norm_dist_prob(pi_star[0]) / norm_dist_prob(pi[t - 1])))   #alpha值

    u = random.uniform(0, 1)
    if u < alpha:
        pi[t] = pi_star[0]
    else:
        pi[t] = pi[t - 1]


plt.scatter(pi, norm.pdf(pi, loc=10, scale=5),label='Target Distribution', c= 'red')
num_bins = 50
plt.hist(pi, num_bins, density=1, facecolor='green', alpha=0.7,label='Samples Distribution')
plt.legend()
plt.show()

吉布斯采样（Gibbs Sampling）

接下来，我们看看更现实的应用场景，高维数据的情况。

假设Xt={Xt,1,Xt,2,⋯ ,Xt,d}\boldsymbol{X}_{t}= \{X_{t, 1}, X_{t, 2}, \cdots, X_{t, d}\}Xt​={Xt,1​,Xt,2​,⋯,Xt,d​}是一个ddd维的随机变量。 Gibbs采样每个时刻ttt，只对ddd维中的一维进行采样。 比如说我们接下来要对第hhh维采样，我们将时刻t+1t+1t+1的样本记作Xt+1=Xt,[Xt,h=Xt+1,h]\boldsymbol{X}_{t+1} = \boldsymbol{X}_{t, [X_{t, h}=X_{t+1, h}]}Xt+1​=Xt,[Xt,h​=Xt+1,h​]​ ，我们定义转移概率（这里的多维变量，我们用最稳妥的写法）为

q(Xt+1∣Xt)=P(Xt+1,h∣Xt,−h)=P(Xt,[Xt,h=Xt+1,h])P(Xt,−h),q(\boldsymbol{X}_{t+1} \mid \boldsymbol{X}_{t}) = P(X_{t+1, h} \mid \boldsymbol{X}_{t, -h})= \frac{P(\boldsymbol{X}_{t, [X_{t, h}=X_{t+1, h}]})}{P(\boldsymbol{X}_{t, -h})}, q(Xt+1​∣Xt​)=P(Xt+1,h​∣Xt,−h​)=P(Xt,−h​)P(Xt,[Xt,h​=Xt+1,h​]​)​,

其中Xt,−h=Xt∖{Xt,h}\boldsymbol{X}_{t, -h} = \boldsymbol{X}_{t} \setminus \{X_{t, h}\}Xt,−h​=Xt​∖{Xt,h​}。 这个式子意味着，我们每次固定第hhh维以外的维度的取值，然后只更新第hhh维的取值。 然后，我们定义接受率

At+1,t:=min(1,P(Xt+1)q(Xt∣Xt+1)P(Xt)q(Xt+1∣Xt))=min(1,P(Xt,[Xt,h=Xt+1,h])P(Xt,h∣Xt,−h)P(Xt)P(Xt,h∣Xt,−h))=min(1,P(Xt,[Xt,h=Xt+1,h])P(Xt,h∣Xt,−h)P(Xt,−h)P(Xt)P(Xt,[Xt,h=Xt+1,h]))=min(1,P(Xt,[Xt,h=Xt+1,h])P(Xt)P(Xt)P(Xt,[Xt,h=Xt+1,h]))=min(1,1)=1.\begin{aligned} \mathcal{A}_{t+1, t} &:= \mathrm{min}\left(1, \frac{P(\boldsymbol{X}_{t+1})q(\boldsymbol{X}_{t}\mid \boldsymbol{X}_{t+1})}{P(\boldsymbol{X}_{t})q(\boldsymbol{X}_{t+1}\mid \boldsymbol{X}_{t})}\right) \\ &= \mathrm{min}\left(1, \frac{P(\boldsymbol{X}_{t, [X_{t, h}=X_{t+1, h}]})P(X_{t, h} \mid \boldsymbol{X}_{t, -h})}{P(\boldsymbol{X}_{t})P(X_{t, h} \mid \boldsymbol{X}_{t, -h})}\right) \\ &= \mathrm{min}\left(1, \frac{P(\boldsymbol{X}_{t, [X_{t, h}=X_{t+1, h}]})P(X_{t, h} \mid \boldsymbol{X}_{t, -h})P(\boldsymbol{X}_{t, -h})}{P(\boldsymbol{X}_{t})P(\boldsymbol{X}_{t, [X_{t, h}=X_{t+1, h}]})}\right) \\ &= \mathrm{min}\left(1, \frac{P(\boldsymbol{X}_{t, [X_{t, h}=X_{t+1, h}]})P(X_{t})}{P(\boldsymbol{X}_{t})P(\boldsymbol{X}_{t, [X_{t, h}=X_{t+1, h}]})}\right) \\ &= \mathrm{min}(1, 1) \\ &= 1. \end{aligned} At+1,t​​:=min(1,P(Xt​)q(Xt+1​∣Xt​)P(Xt+1​)q(Xt​∣Xt+1​)​)=min(1,P(Xt​)P(Xt,h​∣Xt,−h​)P(Xt,[Xt,h​=Xt+1,h​]​)P(Xt,h​∣Xt,−h​)​)=min(1,P(Xt​)P(Xt,[Xt,h​=Xt+1,h​]​)P(Xt,[Xt,h​=Xt+1,h​]​)P(Xt,h​∣Xt,−h​)P(Xt,−h​)​)=min(1,P(Xt​)P(Xt,[Xt,h​=Xt+1,h​]​)P(Xt,[Xt,h​=Xt+1,h​]​)P(Xt​)​)=min(1,1)=1.​

Gibbs采样

初始状态为x0\boldsymbol{x}_{0}x0​, t时刻状态的分布为xt\boldsymbol{x}_{t}xt​。
通过如下流程采样样本xt+1\boldsymbol{x}_{t+1}xt+1​:

  1. 均匀的从[d][d][d]中选取一个维度hhh。
  2. 计算q(Xt+1∣Xt)=P(Xt,[Xt,h=Xt,h])P(Xt,−h)q(\boldsymbol{X}_{t+1} \mid \boldsymbol{X}_{t})= \frac{P(\boldsymbol{X}_{t, [X_{t, h}=X_{t, h}]})}{P(\boldsymbol{X}_{t, -h})}q(Xt+1​∣Xt​)=P(Xt,−h​)P(Xt,[Xt,h​=Xt,h​]​)​
  3. 通过q(Xt+1∣Xt)q(\boldsymbol{X}_{t+1} \mid \boldsymbol{X}_{t})q(Xt+1​∣Xt​)采样第hhh维的一个状态xt+1,hx_{t+1, h}xt+1,h​。
  4. xt+1←xt,[xt,h=xt+1,h]x_{t+1} \gets \boldsymbol x_{t, [x_{t, h}=x_{t+1, h}]}xt+1​←xt,[xt,h​=xt+1,h​]​。

最后一步，是将第ddd维的数据替换成新采样的状态。

import random
import math
from scipy.stats import beta
from scipy.stats import norm
import matplotlib.pyplot as plt

from mpl_toolkits.mplot3d import Axes3D
from scipy.stats import multivariate_normal

samplesource = multivariate_normal(mean=[5,-1], cov=[[1,0.5],[0.5,2]])

def p_ygivenx(x, m1, m2, s1, s2):
    return (random.normalvariate(m2 + rho * s2 / s1 * (x - m1), math.sqrt(1 - rho ** 2) * s2))

def p_xgiveny(y, m1, m2, s1, s2):
    return (random.normalvariate(m1 + rho * s1 / s2 * (y - m2), math.sqrt(1 - rho ** 2) * s1))

N = 5000
K = 20
x_res = []
y_res = []
z_res = []
m1 = 5
m2 = -1
s1 = 1
s2 = 2

rho = 0.5
y = m2

for i in range(N):
    for j in range(K):
        x = p_xgiveny(y, m1, m2, s1, s2)   #y给定得到x的采样
        y = p_ygivenx(x, m1, m2, s1, s2)   #x给定得到y的采样
        z = samplesource.pdf([x,y])
        x_res.append(x)
        y_res.append(y)
        z_res.append(z)

num_bins = 50
plt.scatter(x_res, norm.pdf(x_res, loc=5, scale=1),label='Target Distribution x', c= 'green')
plt.scatter(y_res, norm.pdf(y_res, loc=-1, scale=2),label='Target Distribution y', c= 'orange')
plt.hist(x_res, num_bins, density=1, facecolor='Cyan', alpha=0.5,label='x')
plt.hist(y_res, num_bins, density=1, facecolor='magenta', alpha=0.5,label='y')
plt.title('Histogram')
plt.legend()
plt.show()
